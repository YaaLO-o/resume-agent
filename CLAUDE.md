# Resume Agent — Agent Guidance

面试复盘积累系统。四模式闭环：面试复盘 → 结构化积累 → 岗位匹配 → 弱点攻克。

## 项目结构

```
resume-agent/
├── skills/resume-agent/SKILL.md   # 核心 skill（知识框架 + 模式路由）
├── commands/                       # 四模式执行文件
│   ├── interview-review.md         # 模式一
│   ├── job-analysis.md             # 模式二
│   ├── batch-process.md            # 模式三
│   └── mistake-practice.md         # 模式四
├── CLAUDE.md                       # 本文件
└── README.md                       # 公开文档
```

## 模式检测

| 信号 | 进入模式 |
|------|----------|
| 包含"面试复盘""面试反馈""被问了""面经" | 模式一：面试复盘 |
| 包含"岗位职责""职位要求""JD" | 模式二：岗位分析 |
| 说"处理待投递""处理待处理岗位""批量" | 模式三：批量处理 |
| 说"错题本""练习模式""弱点练习" | 模式四：错题练习 |
| 多信号或模糊 | 追问确认 |

## 核心约束

**真实性 > JD匹配度 > ATS优化 > 文笔风格**

- 绝不编造经历/数字/技能
- 所有内容可追溯到原始素材
- 信息不足时标注而非编造

## 数据存储

1. **SQLite 是主存储**：结构化面试数据
2. **Markdown 是镜像**：人类可读格式
3. **经历库只读**：只在发现新事实时追加
4. **DB 操作必须事务包裹**
5. **去重**：hash(company + date + question_text)
