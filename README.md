# Resume Agent — 面试复盘积累系统

> 贴简历 → 自动建库 → 贴JD → 自动匹配 → 贴面经 → 自动复盘。你只管贴，剩下的交给 AI。

## 它能干什么

把每次面试变成**可检索、可追踪、可改进**的结构化资产。不生成完整简历——专注积累与备战。

| 模式 | 你做什么 | AI 自动做什么 |
|------|---------|-------------|
| **导入简历** | 贴简历文本/上传文件 | 解析所有经历 → 生成 STAR 原子块 → 建立经历库 |
| **面试复盘** | 贴面经/复盘文本 | 解析问答 → 评估质量 → 生成三段反馈 → 检测错题候选 |
| **岗位分析** | 贴 JD | 四维度匹配经历 → 写简历Bullet → 生成补齐方案/项目复盘/思维训练/面试准备 |
| **批量处理** | 说"处理待处理岗位" | 自动遍历所有待投递岗位，跑通全流程 |
| **错题练习** | 说"错题本" | 聚类弱点 → 生成 v1/v2/v3 三个版本答案 |

## 安装

### Claude Code（终端）

```bash
claude plugin marketplace add YaaLO-o/resume-agent
claude plugin install resume-agent@resume-agent
```

### Claude Cowork（桌面端）

1. 打开 **Customize**（左下角）
2. 进入 **Browse plugins** → **Personal** → **+**
3. 选 **Add marketplace from GitHub**
4. 输入：`YaaLO-o/resume-agent`

## 快速上手

### 第一步：贴简历，自动建库

安装完成后，直接贴你的简历（支持中文/英文简历，支持粘贴文本或上传 .txt/.md/.docx/.pdf 文件）：

```
（粘贴你的简历全文）
```

AI 自动完成：
- 解析个人信息、教育背景、实习/项目/竞赛/校园经历
- 生成 STAR 格式的经历原子块
- 推断求职方向
- 建立完整的经历库

> **不支持音频文件**。如需导入录音转写的面经，请先转为文字再贴入。

### 第二步：贴 JD，自动匹配

```
岗位职责：负责xxx...任职要求：xxx...
```

AI 自动完成：
- 四维度匹配你最相关的经历
- 写出可直接用于简历的 Bullet
- 生成补齐方案、项目复盘、思维训练、面试准备

### 第三步：贴面经，自动复盘

```
今天面了xxx，面试官问了xxx，我回答了xxx...
```

AI 自动完成：
- 解析所有问答对
- 评估回答质量
- 生成三段反馈
- 检测是否需要加入错题本

### 第四步：错题攻克

```
错题本
```

AI 自动完成：
- 聚类你的弱点领域
- 为高频卡壳问题生成 v1/v2/v3 三个版本答案

## 工作原理

```
贴简历 → Claude 自动解析 → 建立经历库（markdown，SQLite 可选）
  → 贴 JD → 自动检索经历库 → 四维度匹配 → 生成 Bullet + 备战方案
  → 面试 → 贴面经 → 自动复盘 → 弱点存入错题本
  → 下次分析岗位时，自动交叉匹配历史弱点 → 生成备战警告
  → 循环，每次复盘都在喂养下一次面试准备
```

## 五个命令

| 命令 | 说明 |
|------|------|
| `/resume-agent:import-resume` | 导入简历：解析简历 → 自动建立经历库 |
| `/resume-agent:interview-review` | 面试复盘：解析面经 → 结构化存储 → 三段反馈 |
| `/resume-agent:job-analysis` | 岗位分析：JD解析 → 经历匹配 → 简历Bullet → 补齐方案 → 项目复盘 → 思维训练 → 面试准备 |
| `/resume-agent:batch-process` | 批量处理：读取投递追踪表 → 对每条待处理岗跑通全流程 |
| `/resume-agent:mistake-practice` | 错题练习：弱点聚类 → 答案版本管理 |

## 个性化配置

### 填写你的信息

编辑 `skills/resume-agent/SKILL.md` 中的「候选人信息」部分，把占位符替换为你的真实信息：

```markdown
## 候选人信息

- {你的姓名}，{你的学校} {你的专业} {你的学历} {你的届别}
- 求职方向：{方向1}、{方向2}
- 核心经历：{经历1} + {经历2} + {经历3}
```

> **推荐**：直接用「导入简历」命令，AI 自动填写以上信息。

### 添加你的经历

在 `database/` 目录下编辑对应的 markdown 文件：
- `internships.md` — 实习经历（STAR 格式）
- `projects.md` — 项目经历
- `competitions.md` — 竞赛获奖
- `leadership.md` — 社会实践
- `education.md` — 教育背景 + 课程
- `skills.md` — 技能清单（按求职方向分组）
- `master-resume.md` — 简历总览（AI 自动生成）

> **推荐**：直接用「导入简历」命令，AI 自动创建以上全部文件。

## Excel 模板（可选增强）

本包自带 3 个空白 Excel 模板：

| 模板 | 用途 | 包含 Sheet |
|------|------|-----------|
| `templates/投递追踪模板.xlsx` | 投递管理主表 | 投递追踪、简历Bullet、补齐方案、项目复盘、思维训练、面试准备、学习计划 |
| `templates/复盘导入模板.xlsx` | 面试复盘批量导入 | 问答级导入（公司/岗位/日期/轮次/问题/类型/质量/卡壳/标签） |
| `templates/复盘追踪模板.xlsx` | 岗位复盘追踪 | 匹配度评分、弱点标签、经历映射策略 |

### 使用方式

```
第 1 步：复制模板到工作目录
  templates/投递追踪模板.xlsx → 投递追踪.xlsx（重命名去掉"模板"）

第 2 步：在「投递追踪」sheet 填写：公司名 + 岗位名 + JD原文

第 3 步：对 Claude 说"处理待处理岗位"

第 4 步：AI 自动跑通全流程，回填所有 sheet
```

**你只需要填公司名 + 岗位名 + JD原文。** 其他所有列（岗位类型、匹配方向、主力经历、简历Bullet、补齐方案、项目复盘……）全部由 AI 自动解析填充。

| 你需要填的 | AI 自动完成的 |
|-----------|-------------|
| 公司名 | 岗位类型判定 |
| 岗位名 | 匹配方向分析 |
| JD 原文 | 主力经历推荐 |
| — | 简历 Bullet（含公司+岗位标注） |
| — | 能力缺口 + 补齐方案 |
| — | 项目复盘（灵魂三问+三种时长版本） |
| — | 产品思维训练 |
| — | 面试准备清单 |
| — | 学习计划 |

### 批量处理限制

- 每次最多处理 **10 个岗位**（防止长时间运行）
- 单个岗位处理失败不影响其他岗位
- 处理前会先检查是否已有导入的简历

## 数据存储

- **默认**：所有数据以 Markdown 文件存储在 `database/` 目录，人类可读
- **可选增强**：配置 SQLite 后，结构化数据写入数据库，查询更快
- **模板 vs 工作文件**：`templates/` 下是空白模板，使用时需复制到工作目录并重命名（去掉"模板"二字）
